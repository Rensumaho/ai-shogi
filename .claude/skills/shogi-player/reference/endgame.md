# 終盤リファレンス

終盤の寄せ方・詰み手筋・囲い崩しをまとめる。
終盤の方針決定・指し手選択時に参照する。

---

## 終盤の判断基準

以下のいずれかに該当したら「終盤」に入ったと判断する:

- どちらかの囲いが攻められ始めた
- 玉の周囲に相手の駒が迫っている
- 王手がかかった
- 大駒（飛車・角）の交換が行われ、打ち込みのリスクが生じている

---

## 寄せの基本方針

### 最重要公式: 詰み = 逃げ道封鎖 + 頭金

1. まず相手玉の逃げ道を塞ぐ
2. そのあと玉の頭（1マス上）に金を打つ
3. この順番が逆だと詰まない

### 5つの原則

1. **逃げ道を封鎖してから王手** — 直接王手は逃げられやすい。先に退路を断つ
2. **玉は包むように寄せよ** — 一方向の攻めは逃げられる。複数方向から圧力をかける
3. **守り駒を先に剥がす** — 玉を直接攻めるより、玉の周囲の金銀を取る方が効率的
4. **玉を下段に落とす** — 下段の玉は少ない駒で詰みやすい
5. **金はトドメに残す** — 持ち駒の金は最後の詰みに使う。途中で無駄遣いしない

## 基本的な詰み手筋

### 頭金（あたまきん）

玉の1マス上に金を打つ。最も基本的な詰みの形。

```
（例: 相手玉が5一にいる場合）
  ５
| 金|二 ← ここに金を打つ（▲5二金打）
| 玉|一
```

**なぜ詰むか**: 金は前3方向+横2方向に利くため、玉が横に逃げられない。上にも金がいて逃げられない。下は盤の端。

**条件**: 玉の上のマスに味方の駒がなく、相手の利きもない（金を取られない）こと。

---

### 腹金（はらきん）

玉の横に金を打つ手筋。下段の玉に有効。

```
（例: 相手玉が2九にいる場合）
| 金| 玉|九 ← 玉の横に金を打つ
```

**使い所**: 下段に追い詰めた玉に対して横から金を打つ。

---

### 尻金（しりきん）

玉の真下（後方）に金を打つ手筋。

```
| 玉|一
| 金|二 ← 玉の下に金を打つ
```

**使い所**: 上段の玉に対して下から圧力をかける。

---

### 腹銀（はらぎん）

玉の斜め前に銀を打つ手筋。複数の詰み筋を同時に作る。

```
  ５ ４
|   | 銀|二 ← ここに銀を打つ
| 玉|   |一
```

**使い所**: 直接的な王手ではなく「詰めろ」（次に詰む形）をかける手として有効。

---

### 桂打ちの詰み

桂馬は他の駒を飛び越えるため、合駒が利かない王手になる。

```
（例: 相手玉が5一にいる場合、4三に桂を打つ）
桂馬が5一の玉に王手 → 間に駒を置いても防げない
```

**使い所**: 合駒の利かない王手は非常に強力。終盤では桂馬の持ち駒を大事にする。

---

## 詰めろと必至

### 詰めろ

「次に詰みますよ」という状態。相手は受けないと詰まされる。

**例**: 自分の竜が相手玉の近くにいて、次に▲X金打で詰む形。

### 必至（ひっし）

「どう受けても詰む」状態。詰めろの最強版。必至がかかれば勝ち確定。

**方針**: 直接詰ますのが難しい場合、まず必至を目指す。

## 囲い崩しの手筋

### 矢倉崩し

矢倉の弱点は横と端。

- **横から飛車**: ▲8一飛 or 竜で横から攻める
- **端攻め**: ▲1五歩から端を突き崩す。△同歩▲同香で端の突破を狙う
- **6六への桂打ち**: 金銀の連結を断つ

### 美濃崩し

美濃の弱点は端と上部。

- **端攻め**: ▲1五歩△同歩▲1三歩△同香▲1四歩 と端を破る
- **コビン攻め**: 3六の地点（角のライン）から攻める
- **5筋からの攻め**: ▲5五角や▲5三歩打で上から圧力
- **玉頭攻め**: ▲2四歩から玉頭を直接攻める

### 穴熊崩し

穴熊は非常に堅いため、特別な手筋が必要。

- **大駒を切る**: 飛角を捨ててでも金銀を剥がす（穴熊は金銀が剥がれると一気に弱い）
- **と金攻め**: 穴熊の近くでと金を作り、ゆっくり剥がしていく
- **Z（ゼット）**: 自玉が絶対に詰まない状態を確認してから、持ち駒を全部使って攻める

## スピード計算

終盤はスピード勝負。自分の攻めと相手の攻めの速度を比較する。

### 比較の仕方

1. **自分が相手玉を詰ますのに何手かかるか**を数える
2. **相手が自玉を詰ますのに何手かかるか**を数える
3. 自分の方が速ければ攻める。遅ければ受ける。

### 速度の目安

| 状態 | 手数の目安 |
|------|-----------|
| 詰み | 0手（即詰み） |
| 1手詰めろ | 1手（次に詰む） |
| 2手すき | 2手（受けないと2手後に詰む） |
| 3手すき | 3手（余裕あり） |

### 判断基準

- 自分の攻めが相手より**1手以上速い** → 攻め合い勝ち。攻め続ける
- 自分の攻めが相手と**同速** → 手番を持っている方が有利。王手をかけて手番を握る
- 自分の攻めが相手より**遅い** → 受けに回る。相手の攻めを遅らせてから反撃

---

## 終盤で意識すること

- **無駄な王手をしない**: 王手は逃げられると1手損。逃げ道を塞ぐ方が価値が高い
- **駒を渡しすぎない**: 攻めに使った駒は相手の持ち駒になる。反撃のリスクを考える
- **自玉の安全確認**: 攻める前に「自玉は大丈夫か？」を必ず確認する
- **詰みがあるなら最優先で詰ます**: 詰みが見えたら他の手は考えない

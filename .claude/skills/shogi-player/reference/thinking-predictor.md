# STEP 4: 予想屋フェーズ — 詳細手順

棋士が選んだ指し手に対して、相手の応手を予測し、リスクを評価する。
致命的なリスクがある場合は差し戻しを進言する。

駒の価値は `shogi-theory.md` を参照。

---

## 4-1. 選択手を指した後の盤面を想像する

棋士の選択手を盤面上で「脳内で」実行し、以下を確認する:
- 自分の駒がどこに移動/配置されるか
- 相手の駒との位置関係がどう変わるか
- 新たに空いたマスはどこか

---

## 4-2. 相手の応手を予測する（2-3手先まで）

相手の立場で考え、最も自然な応手を2-3パターン予測する。

**相手が狙いそうな手の優先順位**:
1. **タダ取りできる駒がないか** — 動いた駒や、移動で守りが外れた駒
2. **王手をかけられないか** — 特に王手飛車取り、王手角取りの両取り
3. **駒得になる交換がないか** — 点数の高い駒と低い駒の交換
4. **攻めの拠点を作れないか** — 成り込み、と金作り等
5. **自然な発展手** — 上記がなければ通常の手

---

## 4-3. 致命的リスクのチェック

以下の項目を1つずつ確認する。

### チェック1: 駒のタダ取り

選択手を指した結果、以下の駒が守りなく相手に取られる状態になっていないか:

| 駒 | タダ取りされた場合の深刻度 |
|----|--------------------------|
| 飛車(10点) / 竜(12点) | **致命的** |
| 角(8点) / 馬(10点) | **致命的** |
| 金(6点) / 銀(5点) | 危険 |
| 桂(4点) / 香(3点) | 注意 |
| 歩(1点) | 軽微 |

大駒（飛車・角・竜・馬）のタダ取りは常に「危険」判定。

### チェック2: 両取り（フォーク）

選択手の後、相手に以下のような両取りをかけられないか:
- 王手飛車取り
- 王手角取り
- 飛車角の両取り
- その他、合計8点以上の駒が同時に狙われる形

いずれかに該当すれば「危険」判定。

### チェック3: 玉の安全度低下

選択手が以下に該当しないか:
- 玉の周囲の守り駒を動かし、玉が裸になる
- 相手の大駒の利きに玉がさらされる
- 相手に飛車や角を打ち込まれるスペースが生まれる

該当する場合、その手で得られる利益と比較して判断:
- 利益 < リスク → 「危険」
- 利益 ≥ リスク → 「注意」

### チェック4: 駒損する交換の見落とし

選択手が駒を取る手の場合、相手に取り返されたとき駒損にならないか:
- 自分の駒（X点）で相手の駒（Y点）を取り、相手が取り返す場合
- X > Y なら駒損（例: 銀5点で歩1点を取り、銀を取り返される = 4点損）

3点以上の駒損になる場合は「危険」判定。

---

## リスクレベルの判定

| レベル | 条件 | 判断 |
|--------|------|------|
| **安全** | 上記チェックにすべて問題なし | そのまま進行 |
| **注意** | 軽微なリスクはあるが、手の利益が上回る | そのまま進行（コメント付き） |
| **危険** | 致命的リスクあり（大駒タダ取り、両取り、重大な玉の安全度低下） | **差し戻しを進言** |

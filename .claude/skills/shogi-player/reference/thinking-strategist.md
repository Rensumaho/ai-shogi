# STEP 2: 戦略家フェーズ — 詳細手順

盤面情報と知識ベースを基に、現在の局面を分析し方針を策定する。

**参照すべき知識ベース**（同ディレクトリ内）:
- `shogi-theory.md` — 駒の価値・形勢判断（常に参照）
- `castles.md` — 囲いの選択・手順（序盤で参照）
- `attacks.md` — 攻め筋の選択・手順（序盤・中盤で参照）
- `endgame.md` — 寄せ方・詰み手筋（終盤で参照）

---

## 2-1. 局面段階の判定

| 段階 | 判定基準 |
|------|----------|
| 序盤 | 手数が20手以下、かつ駒の交換がほとんど発生していない |
| 中盤 | 駒の交換が始まっている、両者に持ち駒がある |
| 終盤 | どちらかの囲いが崩れている、または玉の周囲に敵駒が迫っている |

王手がかかっている場合は常に「終盤」扱いとする。

---

## 2-2. 形勢評価

`shogi-theory.md` の駒の価値表に基づき計算する:

```
自分の戦力 = 盤上の自駒の点数合計 + 持ち駒の点数合計
相手の戦力 = 盤上の相手駒の点数合計 + 相手持ち駒の点数合計
形勢差 = 自分の戦力 − 相手の戦力
```

| 形勢差 | 判定 |
|--------|------|
| +5以上 | 大きく有利 |
| +2〜+4 | やや有利 |
| -1〜+1 | 互角 |
| -4〜-2 | やや不利 |
| -5以下 | 大きく不利 |

駒の点数に加え、以下も加味する:
- 成駒の有無（竜・馬は特に価値が高い）
- 遊び駒の有無（働いていない駒は実質的に戦力外）
- 玉の安全度（囲いが健在か崩れているか）

## 2-3. 方針の策定

局面段階と形勢に応じて方針を決める。

### 序盤の方針

| 状況 | 方針 |
|------|------|
| 初手〜数手 | 飛車先の歩を突き、戦型を選ぶ（居飛車 or 振り飛車） |
| 戦型決定後 | 玉を囲う。飛車と反対側に金銀3枚で守りを固める |
| 囲い完成 | 攻めの形を作る（棒銀、角道を開ける等） |

居飛車の場合:
- 先手なら飛車先を伸ばし、相手の出方を見て棒銀 or 矢倉を選択
- 後手なら相手の戦型に合わせて対応

### 中盤の方針

| 形勢 | 方針 |
|------|------|
| 有利 | 駒交換を積極的に行い、リードを広げる。大駒の成り込みを狙う |
| 互角 | 相手の弱点を探しつつ、自陣の守りも忘れない。駒損しない攻めを心がける |
| 不利 | 無理攻めを避け、守りを固める。相手のミスを待ちカウンターを狙う |

共通: 大駒（飛車・角）が遊んでいたら活用を最優先する

### 終盤の方針

| 形勢 | 方針 |
|------|------|
| 有利 | 寄せに入る。逃げ道を封鎖し、頭金を狙う |
| 互角 | スピード勝負。相手より先に寄せる手を探す |
| 不利 | 自玉の安全を確保しつつ、逆転の一手を探す |

王手がかかっている場合: 王手の対処が最優先。逃げる/合駒する/取るの3択から選ぶ

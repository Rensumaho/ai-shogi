# 囲いリファレンス

囲いの種類・組み方の手順・特徴をまとめる。
序盤で囲いを選択・構築する際に参照する。

---

## 囲い選択の基本方針

| 自分の戦型 | 推奨囲い | 備考 |
|-----------|---------|------|
| 居飛車 vs 居飛車 | 矢倉、雁木 | 上からの攻めに対応 |
| 居飛車 vs 振り飛車 | 舟囲い → 居飛車穴熊 | 急戦なら舟囲い、持久戦なら穴熊 |
| 振り飛車 | 美濃 → 高美濃 → 銀冠 | 状況に応じて発展させる |

---

## 居飛車の囲い

### 矢倉（金矢倉）

最も基本的な居飛車の囲い。上からの攻めに強い。

**完成形（先手）**:
```
  ９ ８ ７ ６
|   |   | 銀| 金|七（7七銀・6七金）
|   | 玉| 金|   |八（8八玉・7八金）
| 香| 桂|   |   |九
```

**手順（先手の場合）**:
1. ▲7六歩 — 角道を開ける
2. ▲6八銀 — 銀を活用準備
3. ▲7七銀 — 銀を矢倉の位置に
4. ▲7八金 — 金を囲いの位置に
5. ▲5八金 — もう一枚の金を準備
6. ▲6七金右 — 金を矢倉の位置に
7. ▲6九玉 — 玉を移動開始
8. ▲7九玉 — 玉を囲いへ
9. ▲8八玉 — 玉を入城して完成

**手数**: 約9手
**強み**: 上からの攻めに非常に強い、金銀が密集
**弱み**: 横からの攻め（飛車の横利き）にやや弱い、完成まで時間がかかる

---

### 舟囲い

居飛車で振り飛車と戦うときの簡易囲い。急戦向き。

**完成形（先手）**:
```
玉は6八or7八、金は5八周辺
```

**手順（先手の場合）**:
1. ▲5八金右 — 金を中央に寄せる
2. ▲6八玉 — 玉を左に移動
3. ▲7八玉 — さらに左へ（ここまでで最低限の形）

**手数**: 3-4手
**強み**: 非常に素早く組める、攻めに手数を回せる
**弱み**: 守りが薄い、長期戦には不向き

---

### 居飛車穴熊

居飛車で振り飛車に対して使う最強クラスの囲い。

**完成形（先手）**:
```
玉1九（or9九）、香は上げて退避、金銀で蓋
```

**手順（先手の場合、対振り飛車）**:
1. ▲5八金右 — 金を準備
2. ▲6八玉 — 玉を移動開始
3. ▲7八玉 — さらに左へ
4. ▲7九玉 — 下段へ
5. ▲8八玉 — 端に向かう
6. ▲9八香 — 香を上げて退避
7. ▲9九玉 — 玉が穴に入る
8. ▲8八銀 — 銀で蓋をする
9. ▲7八金 — 金で固める

**手数**: 約9手
**強み**: 圧倒的な堅さ、横からの攻めに非常に強い
**弱み**: 完成まで時間がかかる、端攻めに弱い、崩れ始めると一気に崩壊する

---

### 雁木（がんぎ）

矢倉の変化形。6七銀型で玉頭に厚みがある現代的な囲い。

**手順（先手の場合）**:
1. ▲7六歩 — 角道を開ける
2. ▲6八銀 — 銀を準備
3. ▲6六歩 — 角道を止める
4. ▲6七銀 — 銀を雁木の位置に（矢倉の7七銀との違い）
5. ▲5八金右 — 金を準備
6. ▲7八金 — 金を配置
7. ▲6九玉 → ▲7九玉 — 玉を入城

**手数**: 約7-8手
**強み**: 矢倉より手数が少ない、玉頭が厚い、バランスが良い
**弱み**: 8筋の守りが手薄（7七に銀がいない）

## 振り飛車の囲い

### 美濃囲い

振り飛車の基本囲い。横からの攻めに非常に強い。

**完成形（先手・振り飛車の場合）**:
```
  ４ ３ ２ １
|   | 銀| 玉|   |八（3八銀・2八玉）
| 金|   | 桂| 香|九（4九金・2九桂・1九香）
```
※ 左金（6九→5八）も連結して配置するのが本美濃の完成形

**手順（先手・四間飛車の場合）**:
1. ▲7六歩 — 角道を開ける
2. ▲6八飛 — 飛車を四間に振る
3. ▲4八玉 — 玉を右に移動開始
4. ▲3八玉 — さらに右へ
5. ▲2八玉 — 玉を端に寄せる
6. ▲3八銀 — 銀で玉をガード
7. ▲5八金左 — 金を連結

**手数**: 約7手
**強み**: 横からの攻めに非常に強い、素早く組める
**弱み**: 上からの攻め・端攻めにやや弱い

---

### 高美濃囲い

美濃囲いの発展形。上部の守りが強化される。

**美濃囲いからの発展手順**:
1. ▲4六歩 — 金の進出路を作る
2. ▲4七金 — 金を一段上げる

**手数**: 美濃+2手
**強み**: 美濃の横の堅さ+上部の厚み
**弱み**: 金が上がった分、下段がやや手薄

---

### 銀冠

高美濃からさらに発展した形。美濃系の最終形。

**高美濃からの発展手順**:
1. ▲2六歩 — 銀の進出路を作る
2. ▲2七銀 — 銀を上げる（冠の形）
3. ▲3八金 — 金を下段に引いて連結

**手数**: 高美濃+3手（美濃から計5手追加）
**強み**: 上部が非常に厚い、端攻めに強い、玉が逃げやすい
**弱み**: 完成まで時間がかかる、下段（1段目飛車打ち）にやや弱い

**発展の流れ**: 美濃 → 高美濃 → 銀冠（状況を見て段階的に固める）

---

### 振り飛車穴熊

振り飛車で使う穴熊。居飛車穴熊と対抗する堅い囲い。

**手順（先手の場合）**:
1. 美濃囲いの途中まで組む
2. ▲1八玉 — 玉を端に潜り込ませる（居飛車穴熊と逆側）
3. ▲2八銀 — 銀で蓋
4. ▲3八金 — 金で固める

**強み**: 非常に堅い、穴熊同士の戦いに対応できる
**弱み**: 手数がかかる、端攻めに注意

## 囲いの発展・崩壊時の判断

- 囲いが攻められる前に発展できるなら発展する（美濃→高美濃→銀冠）
- 既に攻められているなら囲いの発展より受けを優先する
- 囲いが崩壊し始めたら「終盤」に入ったと判断し、`endgame.md` の寄せに切り替える
